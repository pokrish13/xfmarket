# xfmarket
An analogue of Gnome Software/KDE Discover made for XFCE4 based on GTK3 and Python.
dependencies: python3-gi, gir1.2-gtk-3.0, python3-distro 
Supported distributions: Debian, Arch Linux, openSUSE, Void Linux, Gentoo, Fedora.
Sorry for the poor design and any errors, I'll try to fix them.
Email for suggestions: pokrish.void13@gmail.com
To launch xfmarket:
```bash
git clone https://github.com/pokrish13/xfmarket.git
cd xfmarket
chmod +x xfmarket
./xfmarket
```
